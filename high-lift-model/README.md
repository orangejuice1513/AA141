## High-lift system model (AA141)

This folder is for building a simple, defensible model of wing lift coefficient:

- \(C_L(\alpha,\delta_f,\delta_s)\)
- For two configs: **clean** and **takeoff**

Start here:
- `0_requirements/problem_statement.md`
- `0_requirements/variable_definitions.md`
- `1_data/raw/raymer_ch12/citations.md` (log which Raymer Ch. 12 figures you used)
- `2_model/model_spec.md` (write the model form in words/math before coding)
