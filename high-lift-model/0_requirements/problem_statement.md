## Problem statement

Design a high-lift system concept suitable for a high-aspect-ratio wing and create a performance model that produces:

- \(C_L\) vs \(\alpha\)
- \(C_L\) vs flap deflection \(\delta_f\)
- \(C_L\) vs slat/LE-device deflection \(\delta_s\)
- A combined model \(C_L(\alpha,\delta_f,\delta_s)\)

Configurations:
- **Clean**
- **Takeoff**

Data sources available initially:
- Raymer, *Aircraft Design: A Conceptual Approach*, 6th ed., Chapter 12 figures/graphs

Later data source:
- XFOIL (2D polars for selected airfoil and deflected-flap cases)
